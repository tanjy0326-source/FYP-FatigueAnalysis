# Fatigue-Related Biomechanical Changes in Pickleball Players Using Sit-to-Stand (STS) Movement and Pose Estimation

This repository contains all project materials for my undergraduate FYP at Xiamen University Malaysia (School of Computing & Data Science).

## Contents
- `/docs/FYP_Proposal.pdf` — project background, objectives, and methodology.
- `/src/FYP_2.ipynb` — Python notebook for fatigue analysis using MediaPipe pose estimation.
- `/demo_videos/` — links to sample STS recordings (fresh vs tired).
